함수
자주쓰는 실행 코드를 블럭단위{.지변의성격} 로 묶어서 패키징해놓은 상태

자바와의 차이점 : 파라미터에 함수를 넣을 수 있다.
자주쓰는 코드를 묶어서 기능단위로 재사용 하기 위해서이다
함수를 선언하는 것은 function 예약어를 이용해서 자주 사용할 코드를 묶음으로 처리하는 것
함수의 호출: 정의된 함수를 호출해야 기능이 실행 된다.

함수의 종류

1. 선언적 함수 : 이름을 가진 형태 자바스크립트 파일을 읽을 때 선언적 함수를 가장 먼저 읽게된다
선언적 함수의 호출 위치가 자유롭다 

2. 익명함수 : 함수의 이름이 없는 형태 ex)callback 함수
자체만으로 호출 불가능 변수에 익명함수를 대입 또는 특정이벤트 객체에 대입  호출

3. 대입함수 변수에 익명함수가 대입된 형태가 대입함수 

4. 즉시실행함수   함수가 정의되자마자 실행하는 함수 자주 쓰이지는 않음 ex) 카카오api 지도서비스코드
